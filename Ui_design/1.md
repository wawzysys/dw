### 解决
```
修改 setup.py
打开 setup.py 文件：
使用文本编辑器打开 E:\Program Files (x86)\MATLAB\extern\engines\python\setup.py 文件。

查找并替换版本号：
找到所有包含 version='R2021b' 的行，并将其替换为 version='2021.2'。确保保存文件。
```
